#include <vec3_rtm.hpp>
template<typename value_type, typename vector_type_raw>
move::vectormath::generic_vec3_rtm struct

A three component vector that can store either floats or doubles. The underlying implementation is RTM's vector4f or vector4d.

Template parameters
value_type The type of the components of the vector
vector_type_raw The underlying RTM vector type

Contents

  • Reference
    • Public types
    • Public static variables
    • Public static functions
    • Public functions

Note

Instances of this class must be aligned to 16 bytes.
